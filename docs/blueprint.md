# **App Name**: Cyber Portfolio

## Core Features:

- Terminal Intro: Display a terminal-like intro sequence with typing animation to introduce the user.
- About Section: Display a short bio, skills, and interests in AI, data science, and system engineering styled like a terminal output or JSON config. Implements toggle between "Human Mode" and "Dev Mode".
- Project Showcase: Showcase projects as "missions" or "labs," similar to TryHackMe rooms, with project name, short description, and tech stack badges.
- Skills Dashboard: Present skills using a terminal-based skill matrix or radar chart for categories like Programming, Web Dev, AI/ML, Cloud & DevOps, and Security.
- Experience Timeline: Display experience and education as log entries with timestamps and messages.
- Command-Line Contact: Offer contact information through command-line commands for GitHub, LinkedIn, and email.
- Easter Egg Terminal: Hidden terminal (accessible via Ctrl + ~) allows simple commands like help, about, projects, or skills. Light gamification such as showing a 'Profile Level' or 'XP progress' are available as tools.

## Style Guidelines:

- Primary color: Neon green (#39FF14) to reflect a cyber/hacker theme.
- Background color: Dark grey (#212121) to emulate a terminal interface.
- Accent color: Cyan (#00FFFF) used sparingly for highlights and interactive elements.
- Font: 'Fira Code' monospace font for all text elements, ensuring a code-like appearance. Note: currently only Google Fonts are supported.
- Use icons from Lucide-react to represent various skills and project categories.
- A grid-based layout to ensure a clean and organized display of projects and information.
- Subtle animations for hover effects and transitions, providing a smooth user experience.
